name: voice-draft
description: One-time onboarding to capture the agent's writing voice (email + text) so Realty Stack drafting skills sound like them.
---

Run the voice-draft onboarding workflow. Captures agent profile (name, brokerage, market, signoffs) + 3-5 email samples + 3-5 text samples, analyzes voice, returns summary + 2 example drafts, accepts refinement feedback, and persists to `~/.config/realty-stack/voice-profile.md` on approval.

Full workflow documented at `${CLAUDE_PLUGIN_ROOT}/skills/voice-draft/SKILL.md` — follow it exactly.
